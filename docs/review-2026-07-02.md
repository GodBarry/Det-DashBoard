# 代码与发布审查报告（2026-07-02）

## 结论

当前版本已经具备单机 Ubuntu + Docker Compose 发布条件。LabelMe、标准 COCO、YOLO 的导入和导出主链路、目录浏览、场景筛选、标注编辑及主要项目管理接口均通过自动化测试。默认部署只监听本机，应用使用非 root 用户和只读根文件系统运行。

本次未执行真实模型训练和推理，也未执行跨电脑恢复、备份恢复和多架构实体机验证；这是当前任务明确排除的范围。

## 剩余架构债务

1. `server/postgres-app.js` 仍是大型单体，应按业务域拆分服务与路由，并为数据库事务边界建立统一约定。
2. 数据库 schema 依赖启动时 `ALTER TABLE IF NOT EXISTS`，需要正式的版本化 migration 和回滚策略。
3. 后台任务仍在 API 进程内运行；大规模导入导出应迁移到持久化队列和独立 worker。
4. 当前内部标注只表达矩形框；若要无损支持实例分割，需要引入 polygon/mask 模型和对应编辑器。
5. 视频仅完成资产管理，还缺播放器、抽帧、时间轴、轨迹标注和视频导出协议。
6. 当前单用户边界没有认证。若未来开放局域网或公网，必须先增加认证授权、TLS、CSRF/安全头和更窄的文件系统挂载。
